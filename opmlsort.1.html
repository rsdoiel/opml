NAME

opmlsort

SYNOPSIS

opmlsort OPTIONS INPUT_OPML_FILENAME [OUTPUT_OPML_FILENAME]

DESCRIPTION

opmlsort is a program that sorts the outline in an OPML document.

OPTIONS

-T, -title
sort by title
-ci, -case-insensitive
case insensitive sort
-examples
display examples
-generate-manpage
generate man page
-generate-markdown
generate Markdown documentation
-h, -help
display help
-i, -input
set input filename
-l, -license
display license
-nl, -newline
add trailing newline
-o, -output
set output filename
-p, -pretty
pretty print XML output
-quiet
suppress error messages
-v, -version
display version

EXAMPLES

opmlsort myfeeds.opml sorted-feeds.opml
