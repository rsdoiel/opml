OPML, a Go package and cli

opml

A package to manipulate OPML files in Go. It is based on the spec as described at http://dev.opml.org/spec2.html. It includes a demonstration command line utilities opmlsort which sorts an OPML outline by text attribute values, opmlcat which concatenates one or more OPML outlines and opml2json which converts an OPML file into a JSON document.
