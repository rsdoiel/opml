An OPML parser package plus opml cat and sort utilities

opml

A package to manipulate OPML files in Go. It is based on the spec as described at http://dev.opml.org/spec2.html. It includes a demonstration command line utilities opmlsort which sorts an OPML outline by text attribute values and opmlcat which concatenates one or more OPML outlines.
