opml

A package to manipulate OPML files in Go. It is based on the spec as described at http://dev.opml.org/spec2.html. It includes a demonstration command line utilities

opmlsort
sorts an OPML outline by text attribute values
,opmlcat
Concatenates one or more OPML outlines
opml2json
Converts an OPML file into a JSON document
url2opml
Converts a list of urls into OPML XML
